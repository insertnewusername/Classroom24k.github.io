Rowdy City Wrestling

About the Game

Rowdy City Wrestling is a retro style, arcade wrestling game. The main game mode is the career mode where you fight in 1vs1 matches, royal rumble events and a 6 person free for all brawl mode.

Controls

  • WASD: Movement
  • Spacebar: Jump
  • Shift: Sprint
  • C: Crouch
  • B / E: Shop & Inventory
  • Esc: Exit Fullscreen / Cursor Lock
